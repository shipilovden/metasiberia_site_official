# МетаСибирь — Полный список технических возможностей

## Основная информация

**МетаСибирь (Metasiberia)** — это виртуальный мир, созданный на движке **Glare-core**. Это полнофункциональная 3D-платформа для создания, взаимодействия и совместной работы в виртуальном пространстве.

**Текущая версия**: 0.0.19 (Beta)  
**Дата последнего обновления**: 26 марта 2026 г.  
**Статус**: Pre-release (активная разработка)

---

## 🎮 Основные возможности

### 1. **Виртуальный мир и навигация**
- Полнофункциональная 3D-среда с реалистичной физикой
- Свободная навигация по миру с различными режимами камеры
- Система телепортации между локациями
- Поддержка больших масштабов мира (от комнат до ландшафтов)
- Система координат и парселей (земельных участков)

### 2. **Аватары и персонажи**
- Кастомизируемые аватары пользователей
- Система отслеживания движений (head tracking, body tracking)
- Эмоции и жесты аватаров
- Система имен и профилей пользователей
- Поддержка различных моделей аватаров

### 3. **Общение и социальные функции**
- Встроенный чат (текстовый и голосовой)
- Система сообщений между пользователями
- Групповые беседы и каналы
- Emoji-пикер с категоризацией
- Плавающие сообщения над аватарами
- Система друзей и контактов

### 4. **Создание и редактирование контента**
- **3D-объекты**: загрузка и размещение моделей
- **Текстовые объекты**: 3D-текст с поддержкой шрифтов
- **Изображения**: размещение 2D-изображений в мире
- **Аудио**: встроенные аудиоплеры и звуковые объекты
- **WebView**: встраивание веб-контента в мир
- **Воксели**: система вокселей для создания структур
- **Камеры**: объекты камер для видеозаписи и трансляции
- **Портали**: система порталов для перемещения между мирами

### 5. **Редакторы и инструменты**
- **Object Editor**: редактор объектов с полным контролем параметров
- **Material Editor**: редактор материалов и текстур
- **Parcel Editor**: управление земельными участками
- **World Editor**: редактирование параметров мира
- **Environment Editor**: управление окружением (свет, небо, погода)

### 6. **Система материалов и визуализации**
- Поддержка PBR (Physical Based Rendering)
- Система шейдеров (GLSL)
- Текстурирование и UV-маппинг
- Система освещения (динамическое и статическое)
- Система теней
- Постобработка и эффекты
- Поддержка различных форматов текстур

### 7. **Аудиосистема**
- **Пространственный звук** (Resonance Audio)
- Поддержка форматов: MP3, WAV, OPUS
- Система аудиоплеров
- Микрофонный ввод
- Голосовой чат с кодеками
- Регулировка громкости и эффектов

## 🥽 VR и XR возможности (НОВОЕ в v0.0.19)

### Полная поддержка OpenXR
- **Экспериментальный VR режим** в Windows Qt-клиенте
- **Стерео-рендеринг** с поддержкой VR-шлемов
- **Трекинг головы** (head tracking)
- **Трекинг контроллеров** (controller tracking)
- **Трекинг движений** рук и тела

### Поддерживаемые VR-шлемы
- **Meta Quest 2** (PCVR/OpenXR)
- **Meta Quest 3** (PCVR/OpenXR)
- **HTC Vive** (все версии)
- **HTC Vive Cosmos**
- **HTC Vive Focus 3** (с app-side render models)
- **Valve Index**
- **Microsoft Motion Controllers**
- Khronos generic controller (универсальная поддержка)

### VR-функционал
- **Телепортационная локомоция** (teleport locomotion) с валидацией точек посадки
- **Плавная локомоция** (smooth locomotion) на стиках/трекпадах
- **Recenter-пайплайн** для ручного выравнивания (клавиши Home/End)
- **First-person отображение** аватара в VR
- **Grounding аватара** с привязкой к LeftEye/RightEye/Head
- **Диагностика позы** (xr_pose_trace.csv) для анализа высоты головы, горизонта, симметрии глаз
- **Portal rendering** в VR с корректным tonemapping/bloom
- **Поддержка controller interaction profiles** для различных шлемов

### VR-отладка и утилиты
- **analyze_xr_pose_trace.py** для анализа трассировки позы
- Автоматическая синхронизация записей со шлема через ADB
- Watch-режим для мониторинга
- Bat-стартер для удобного запуска

## 🛠️ Технологический стек

### Языки программирования
- **C++** (47.4% кода) — основной язык ядра
- **C** (46.9% кода) — низкоуровневые компоненты
- **GLSL** (1.5%) — шейдеры
- **CMake** (1.2%) — система сборки
- **Lua** (Luau) — скриптинг и расширения
- **JavaScript** (0.6%) — веб-клиент
- **Fortran** (0.7%) — специализированные вычисления

### Платформы
- **Windows** (основная платформа)
- **Linux** (поддержка)
- **Android** (планируется)
- **Web** (веб-клиент)

### Графика и рендеринг
- **OpenGL** / **DirectX** (в зависимости от платформы)
- **OpenXR** для VR
- **Glare-core** — ядро движка и основа рендеринга

### UI и интерфейс
- **Qt 5 / Qt 6** — desktop-клиент
- **SDL** — кроссплатформенный слой
- **CEF (Chromium Embedded Framework)** — встраивание веб-контента

### Аудио
- **Resonance Audio** — пространственный звук
- **miniaudio** — аудиодвижок
- **minimp3** — MP3-декодер
- **Opus** — кодек для голоса
- **RtAudio** — аудиоинтерфейс

### Сеть и синхронизация
- **Собственный сетевой протокол** для синхронизации состояния
- **Поддержка Ethereum** (интеграция блокчейна)
- **secp256k1** — криптография

### Сборка и развертывание
- **CMake** — кроссплатформенная сборка
- **GitHub Actions** — CI/CD
- **Windows CI** — автоматическая сборка для Windows
- **Linux Configure** — конфигурация для Linux

- **Адаптивная частота** синхронизации

- **Поддержка большого количества** пользователей

## 📝 Заключение

**МетаСибирь** — это амбициозный, полнофункциональный проект виртуального мира с поддержкой VR, создания контента и социального взаимодействия. Проект активно развивается, имеет открытый исходный код и постоянно получает новые функции. Последние обновления (v0.0.16-v0.0.19) сосредоточены на полной интеграции VR через OpenXR, что делает платформу одной из первых русскоязычных VR-метавселенных с профессиональной поддержкой.
