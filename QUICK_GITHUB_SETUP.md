# Быстрая настройка GitHub

```bash
git remote add origin https://github.com/YOUR_USERNAME/metasiberia_site_official.git
git branch -M main
git push -u origin main
```

Важно:

- GitHub здесь только для хранения исходников
- Продакшен публикуется на REG.RU
- Другие старые каналы деплоя для этого проекта не используются
