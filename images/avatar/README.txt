Put horizontal avatar screenshots in this folder.

Supported numbered base names:
- avatars-metasiberia-01
- avatars-metasiberia-02
- avatars-metasiberia-03

Supported extensions:
- .webp
- .png
- .jpg
- .jpeg
- .svg

Examples:
- images/avatar/avatars-metasiberia-01.png
- images/avatar/avatars-metasiberia-02.png
- images/avatar/avatars-metasiberia-03.png

Recommended format:
- horizontal screenshot
- ideally 16:9 or close to it
